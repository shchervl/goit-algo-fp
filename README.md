# goit-algo-fp

Фінальний проєкт курсу з алгоритмів та структур даних (GoIT × Neoversity).

## Налаштування

Проєкт використовує [uv](https://docs.astral.sh/uv/) для керування залежностями.

```bash
uv sync
```

Це створить `.venv/`, встановить залежності та зареєструє виконуваний шорткат `pythagoras-tree`.

## Завдання 1. Структури даних. Сортування. Робота з однозв'язним списком

Реалізовано однозв'язний список з трьома операціями:

1. **Реверсування** — `LinkedList.reverse()`: змінює посилання між вузлами на місці, без створення нових вузлів.
2. **Сортування** — `LinkedList.sort()`: сортування злиттям (merge sort) за O(n log n).
3. **Об'єднання двох відсортованих списків** — `merge_sorted(a, b)`: повертає новий відсортований список без модифікації вхідних.

Файли: `task_01/linked_list.py`, тести в `task_01/tests/test_linked_list.py`.

### Запуск

```bash
# Запуск тестів (77 тестів)
uv run pytest task_01/

# Інтерактивне використання
uv run python -c "
from task_01.linked_list import LinkedList, merge_sorted

ll = LinkedList([3, 1, 4, 1, 5, 9, 2, 6])
ll.sort()
print(ll.to_list())  # [1, 1, 2, 3, 4, 5, 6, 9]

ll.reverse()
print(ll.to_list())  # [9, 6, 5, 4, 3, 2, 1, 1]

a = LinkedList([1, 3, 5])
b = LinkedList([2, 4, 6])
print(merge_sorted(a, b).to_list())  # [1, 2, 3, 4, 5, 6]
"
```

## Завдання 2. Рекурсія. Фрактал «дерево Піфагора»

Дерево Піфагора через рекурсію з інтерактивним UI на tkinter. Підтримує два стилі візуалізації:

- **гілки** — branch-style фрактал (відрізки), симетричний при будь-якому куті
- **квадрати** — класична Піфагорова композиція з квадратами та прямокутними трикутниками (симетрична лише при α=45°)

Структура пакета `task_02/`:

| Модуль | Призначення |
|---|---|
| `geometry.py` | Чисті рекурсивні генератори (`generate_branches`, `generate_squares`, `square_corners`) |
| `styling.py` | Колір і товщина за глибиною |
| `pythagoras_tree.py` | Tkinter UI: радіо-перемикач стилю, повзунки глибини/кута |
| `__main__.py` | Точка входу: запускає `run_ui()` |

### Запуск

```bash
uv run pythagoras-tree
```

Керування у вікні:
- радіо-перемикач **«Стиль»** — *гілки* / *квадрати*
- повзунок **«Рівень рекурсії»** (0–12)
- повзунок **«Кут розгалуження»** (0°–90°)

## Тести

```bash
uv run pytest                # усі тести
uv run pytest task_01/       # тільки задача 1 (77 тестів)
```

Тести покривають task_01: реверс, сортування злиттям, об'єднання, граничні випадки (порожні списки, `None`, різні типи даних: float, bool, кортежі, кастомний клас з `__lt__`, NaN), а також інваріант «реверс саме переставляє посилання, не створюючи нові вузли».
